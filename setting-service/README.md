# # 알림(Notification) 서비스

### 1. 주요 요구사항 정의

#### 1) 기능적 요구사항
- 알림 생성
    - 특정 이벤트 발생 시, 사용자에게 알림 생성
    - 실시간(즉시) 알림 제공
    - 다양한 유형
- 알림 조회
    - 사용자가 자신의 알림을 조회
- 읽음 상태 관리
    - 사용자가 알림 조회 시 읽음 처리
    - 사용자가 알림을 읽었는지 여부 관리

#### 2) 비기능적 요구사항
- 영속성 : 알림 데이터를 안정적으로 저장
- 확장성 : 점진적 사용자 증가 대비 필요
- 성능 : 빠른 읽기/쓰기
- 유지보수성
